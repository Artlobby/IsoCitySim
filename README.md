# IsoCitySim
A simulation of a city using isometric tiles

###How to use IsoCitySim for fun and for profit

1. Clone down the repository and navigate to it

2. `npm install`

3. `bower install`

4. `grunt`

5. Use MAMP or something similar to host the `dist` folder

6. That's it!
